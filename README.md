## 🔍 Project Overview

### **1. Data Preparation (SQL)**
Raw movie data was cleaned and transformed using SQL in Google BigQuery.  
Key steps included:
- Removing invalid or missing values  
- Standardizing budget and revenue fields  
- Creating ROI metrics  
- Extracting release year and genre information  
- Exporting cleaned tables as CSVs for analysis  

### **2. Exploratory Data Analysis (Python)**
Using the cleaned CSVs, the analysis explored:
- Budget vs. revenue relationships  
- ROI distribution across genres  
- Release year trends  
- Rating correlations  
- Outliers and high-performing films  

### **3. Visual Insights**
Visualizations were created to highlight:
- Genre profitability  
- Budget tiers and their ROI  
- Release timing patterns  
- Rating vs. revenue relationships  

### **4. Key Findings**
- Certain genres consistently outperform others in ROI  
- Mid-budget films often deliver stronger ROI than high-budget films  
- Release timing (e.g., summer vs. winter) influences box office performance  
- Ratings correlate with revenue but are not the strongest predictor
